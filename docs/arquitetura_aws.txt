                         FLUXO DE DADOS:
1. Sequenciadores geram arquivos .fasta (dados genômicos)
2. Arquivos são enviados automaticamente para S3 (dados-brutos)
3. Evento de upload dispara a função Lambda
4. Lambda processa os dados e salva resultados em S3 (dados-processados)
5. Metadados são registrados no DynamoDB
6. CloudWatch monitora toda a execução
7. Budgets envia alertas se custo exceder limites
8. EventBridge executa tarefas de manutenção noturnas
